# stikky-cms
A CMS written in NodeJS/Angular.  All CMSes need a purpose, and the purpose of Stikky-CMS is to syndicate content. 

# To run
* npm install
* npm install mysql 
* copy config/config.json.sample to config.json and set settings
* npm run update-db

# To test
* jasmine
